# Streamlit++ Examples

This directory contains comprehensive sample projects demonstrating the capabilities of the Streamlit++ framework. Each example showcases different aspects of the framework's advanced features.

## üìÅ Available Examples

### 1. Basic App (`basic_app.py`)
A simple demonstration of core Streamlit++ features including:
- Enhanced UI components (buttons, cards, badges)
- Basic theming and styling
- Simple animations and layouts

**Run:** `streamlit run examples/basic_app.py`

### 2. Analytics Dashboard (`analytics_dashboard.py`)
A comprehensive business analytics dashboard featuring:
- **Navigation**: Sidebar navigation with multiple pages
- **KPI Dashboard**: Interactive KPI cards with metrics and trends
- **Data Visualizations**: Interactive charts using Plotly
- **User Analytics**: Customer segmentation and behavior analysis
- **Revenue Analytics**: Sales performance and financial metrics
- **Data Tables**: Enhanced data tables with sorting and pagination
- **Animations**: Page transitions and loading states
- **Theming**: Custom color schemes and dark mode support

**Features Demonstrated:**
- Advanced layout management with CSS Grid
- Interactive data visualizations
- Multi-page application structure
- Real-time data filtering
- Responsive design patterns
- Custom component styling

**Run:** `streamlit run examples/analytics_dashboard.py`

### 3. Data Explorer (`data_explorer.py`)
An interactive data exploration application with:
- **Advanced Filtering**: Multi-dimensional data filtering
- **Data Overview**: KPI cards and summary statistics
- **Interactive Exploration**: Customer analysis, product insights, sales analytics
- **Custom Queries**: Dynamic query builder interface
- **Data Export**: Multiple export format options
- **Real-time Updates**: Dynamic data filtering and visualization

**Features Demonstrated:**
- Complex form components
- Advanced data visualization techniques
- Custom query interfaces
- Data export functionality
- Responsive grid layouts
- Interactive filtering systems

**Run:** `streamlit run examples/data_explorer.py`

### 4. Advanced Forms (`advanced_forms.py`)
A sophisticated form application showcasing:
- **Registration Wizard**: Multi-step form with validation
- **Form Components**: Enhanced input controls and selection widgets
- **Real-time Validation**: Custom validation rules and feedback
- **Interactive Survey**: Dynamic survey with progress tracking
- **Data Entry**: File upload and rich text editing

**Features Demonstrated:**
- Multi-step form workflows
- Advanced validation systems
- Custom form components
- Progress tracking and state management
- File upload handling
- Complex form layouts

**Run:** `streamlit run examples/advanced_forms.py`

## üöÄ Getting Started

### Prerequisites
- Python 3.8+
- Streamlit 1.28+
- Required dependencies (install via pip):
  ```bash
  pip install streamlit plotly pandas numpy
  ```

### Running Examples
1. Navigate to the project root directory
2. Run any example using Streamlit:
   ```bash
   streamlit run examples/[example_name].py
   ```
